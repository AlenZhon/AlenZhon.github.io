美团又一面的二面

发表于 2020-04-16 更新于 2020-04-17 分类于 面试

4.16 美团二面 50分钟

  1. 自我介绍
  2. 掰扯项目（项目背景，目标，实现难点，怎么解决）
  3. 平常用过什么数据结构(我有点想往map hashMap上引但是失败了)
  4. 说到了栈，写一个用栈实现队列的算法(leetcode 225 Implement Queue using two stacks)
  5. 二叉树的左右翻转 非递归版本 (leetcode 226 invert binary tree)
  6. 如果有40亿个邮箱没有经过排序，现在给一个邮箱，判断在不在这40亿个邮箱里，用什么数据结构？
    (说了分治+hash数组 B+树，问有没有更好的？有没有了解过位图) （哦豁）
  7. B+树和B树的区别
  8. 唯一索引和主键索引的区别
  9. 再写个题目 合并有序链表(leetcode 21 merge two sorted list)
  10. 类加载过程的了解？ （哦豁）
  11. 线程有哪些状态？ IO阻塞的时候转到blocked还是wait（只能说记不太清了）？
  12. wait和sleep的区别
  13. 死锁的理解？死锁的四个条件总结一下
  14. GC算法？survivor区为什么有两个？用来解决什么问题？
  15. MySQL实现一个乐观锁？ (哦豁)
  16. 10升水 3升桶和7升桶 量出5升水。
  17. 最近在看的一本书是什么？
  18. 说一个你的优缺点
